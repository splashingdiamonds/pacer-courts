Effective October 19, 2022, the new dial-in number for hearings before The United States Bankruptcy Court for the Eastern District of Louisiana is:
1-504-517-1385, Conference Code 129611.
Please use the new phone number and conference code on notices of hearing for all hearings set on and after October 19, 2022. Effective October 19, 2022, the old number and old code will no longer function.

Additionally, as of October 19, 2022, GoToMeeting will replace Zoom. Please use https://gotomeet.me/JudgeGrabill for any video hearings.

Please ensure that you update any Notice of Hearing filed for any hearing set on or after October 19, 2022. Any hearings which have already been set, will be adjusted by order of the Court.


Please see Amended General Order 2021-2 Implementing Hybrid Model Plan for Conduct of Hearings: https://www.laeb.uscourts.gov/sites/laeb/files/AmendedGeneralOrder2021-2reHybridModelPlanforHearings.pdf
